# TheWudlands
An old-school round-based Fighting Fantasy-style adventure game inspired by UltraQuest, EverQuest, books from Steve Jackson, Ian Livingstone, and the GAVUN WUD meme, built as a browser-based fantasy RPG with scene-driven gameplay, pixel-art, ascii-art, narrative-driven adventures, and onchain character progression. The game will be built using a modular, plugin-based architecture that allows for easy extension and modification.
